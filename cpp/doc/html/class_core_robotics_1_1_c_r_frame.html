Detailed Description

This class implements a basic homogeneous transformation made up of rotations and translations.

Description

CRFrame implements a homogeneous transformation specified by a rotation matrix and a translation vector . The transformation matrix

transforms a point in the child frame (i) to a point in the parent frame (i-1) by the operation

Example

This example creates a frame class.

#include "CoreRobotics.hpp"
#include <stdio>
main() {
CoreRobotics::CRFrame Frame;
Eigen::Matrix3d Rot;
Eigen::Vector3d Trans;
Rot << 0, -1, 0, 1, 0, 0, 0, 0, 1;
Trans << -2, 2, 3;
Frame.setRotationAndTranslation(Rot, Trans);
Eigen::Matrix4d T;
Frame.getTransformToParent(T);
std::cout << "Transformation to parent\n" << T << std::endl;
Eigen::Vector3d p, y;
p << 5, 6, 7;
Frame.transformToParent(p, y);
std::cout << "Point " << p.transpose() << " transformed to "
<< y.transpose() << std::endl;
}
