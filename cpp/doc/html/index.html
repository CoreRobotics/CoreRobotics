Introduction

The CoreRobotics libraries provide packages for real-time robot control including kinematics, estimation, percetion, modeling and interfacing with items in the world. The libraries are written in C++ and are intended to be cross-platform, although they have currently only been tested on Windows 8.1 and Mac OS X platforms. The code is distributed under the BSD 3-clause license for flexibility of use.
